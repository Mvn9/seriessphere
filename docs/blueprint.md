# **App Name**: Series Sphere

## Core Features:

- Custom Watchlists: Allow users to organize their favorite shows into categories such as 'Watching', 'Completed', 'On Hold', 'Dropped', and 'Plan to Watch'.
- Episode Tracking: Users can log watched episodes and track progress. Also displays remaining episodes in the series.
- Spoiler Protection: Users can choose to hide/reveal spoilers for shows that they are currently watching.
- AI Recommendations: Utilize an AI tool to make personalized recommendations based on viewing history.
- Custom Clubs & Discussion Boards: Users can create/join groups to discuss shows.

## Style Guidelines:

- Primary color: Soft teal (#4DB6AC) to create a calm and focused viewing atmosphere.
- Secondary colors: Light grey (#EEEEEE) for backgrounds and a darker grey (#616161) for text to ensure readability.
- Accent: Use a vivid purple (#9C27B0) for highlights and interactive elements.
- Clean and modern fonts should be used to ensure easy readability.
- Use consistent, minimalist icons throughout the interface to represent categories and actions.
- The layout should be clean, intuitive, and responsive to ensure ease of use across different devices.
- Subtle transitions and animations to enhance the user experience without being distracting.
